# StocksApp

## English

### Description
This project was created as part of an ASP.NET Core course on Udemy.
The application allows buy and sell stocks and save all orders to PDF file format.
The project was developed for educational purposes to practice working with ASP.NET Core and web application development.

### Features
- Buy stocks
- Sell stocks
- View orders
- View stocks
- Export order data to PDF file format

### Technologies
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- HTML
- CSS
- JavaScript

### Architecture
The project is built using the MVC (Model-View-Controller) pattern.
Entity Framework Core is used for database operations.

### Project Status
Educational project created during a Udemy course.

### Notes
Authentication and authorization are not implemented in this version of the project.
There is also another version of this project developed using Clean Architecture principles.

# StocksApp

## Опис
Цей проєкт був створений у рамках курсу з ASP.NET Core на Udemy.  
Додаток дозволяє купувати та продавати акції, а також зберігати всі замовлення у форматі PDF.  
Проєкт розроблений з освітньою метою для практики роботи з ASP.NET Core та розробки вебзастосунків.

## Функціональність
- Купівля акцій
- Продаж акцій
- Перегляд замовлень
- Перегляд акцій
- Експорт даних замовлень у PDF-файл

## Технології
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- HTML
- CSS
- JavaScript

## Архітектура
Проєкт побудований з використанням патерну MVC (Model-View-Controller).  
Для роботи з базою даних використовується Entity Framework Core.

## Статус проєкту
Навчальний проєкт, створений під час проходження курсу на Udemy.

## Примітки
У цій версії проєкту не реалізовано автентифікацію та авторизацію.  
Також існує інша версія цього проєкту, розроблена з використанням принципів Clean Architecture.
